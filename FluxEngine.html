Flux Engine

Info

Genre Vulkan Renderer
Release Date May 2022
Engine Custom (Vulkan)
Programming Language C++
My Role Graphics Programmer
Development Time 7 months

Features

  • Raymarched volumetric lighting with configurable albedo and density
  • Real-time shadows implemented using variance shadow mapping
  • ImGui integration with Vulkan in engine
  • Model and material loading functionality using Assimp
  • Various engine work and abstractions around Vulkan
