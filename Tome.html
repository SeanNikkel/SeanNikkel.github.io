Tome Voxel Engine

Tome

Info

Genre Exploration
Release Date 2020
Engine Unreal Engine 4
Programming Language C++
My Role Sole Developer
Development Time 1 month

Features

  • Infinite library generation using simplified Wave Function Collapse algorithm
  • Books generated with random content, color, widths, and lengths
  • Books can seamlessly transform to and from physics objects
  • Interactions are smoothed out using interpolation curves
  • Created all models used in game with Blender
